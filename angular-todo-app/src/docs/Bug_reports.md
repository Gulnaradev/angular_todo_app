# BUG REPORTS — Angular Todo App

## BR-01 — Изменённое название задачи не сохраняется после редактирования

**Summary:**  
Edited todo title is not saved after clicking Save

**Environment:**  
- Application: Angular Todo App  
- Browser: Chromium/Firefox/WebKit (Playwright)  

**Preconditions:**  
- Приложение запущено   

**Steps to Reproduce:**
1. Создать новую задачу
2. Нажать кнопку **Edit** у созданной задачи
3. Изменить название задачи в поле ввода
4. Нажать кнопку **Save**

**Actual Result:**  
- После сохранения отображается старое значение вместо нового

**Expected Result:**  
- Изменённое название задачи должно сохраняться и отображаться в списке задач после нажатия **Save**

**Severity:**  
Major — основной пользовательский сценарий (редактирование) не работает корректно

**Priority:**  
High — дефект влияет на ключевой функционал приложения

**Notes:**  
- Дефект воспроизводится стабильно  
- Возможная причина: некорректная обработка значения input при сохранении или отсутствие обновления состояния задачи после PUT-запроса
